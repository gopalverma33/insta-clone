# Insta Clone (Full Stack Project)

This is a simple Instagram-like full stack application created as part of a recruitment task.

The project includes user authentication, post creation, and a basic feed system.

---

## Tech Stack Used

Frontend:
- Next.js
- React
- JavaScript

Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication

---

## Features Implemented

- User Signup and Login
- JWT based authentication
- Token stored in browser
- Home feed showing posts
- Create post using image URL and caption
- Protected routes (only logged-in users can create posts)
- Basic profile page structure

insta-clone/
├── backend/
├── insta-frontend/


---

## How to Run the Project

### Backend

cd backend
npm install
npm start


Runs on:
http://localhost:5000


### Frontend


cd insta-frontend
npm install
npm run dev


Runs on:


http://localhost:3000


---

## Notes

- Image upload is handled using image URLs
- `.env`, `node_modules`, and build files are ignored using `.gitignore`
- Project is kept simple and easy to understand
